1. МОНИТОРИНГ (20 мин)
   ├── Проверка Talkwalker alerts
   ├── Проверка hnrss.org (посты > 50 очков)
   ├── Проверка Reddit RSS (горячие посты)
   ├── Сбор трендовых тем
   └── Генерация черновиков комментариев

2. КОНТЕНТ (40 мин)
   ├── Адаптация pillar-статьи под каждую платформу
   ├── Генерация X тредов и твитов (RU)
   ├── Генерация Threads постов (RU)

   ├── Генерация TG поста (RU)

   ├── Dev.to, Hashnode → curl
   ├── Threads, X, Bluesky, Mastodon → API
   ├── Telegram → MCP
   └── GitHub Gists → API

4. ПУБЛИКАЦИЯ - Playwright (30 мин, ПОСЛЕДОВАТЕЛЬНО)
   ├── LinkedIn → пост
   ├── Substack → Notes + Newsletter
   ├── VC.ru → статья
   └── Хабр → статья

5. ENGAGEMENT - Playwright (20 мин, ОСТОРОЖНО)
   ├── LinkedIn: 5 комментариев
   ├── X: 10 replies
   ├── Medium: 10-20 claps
   └── Reddit: 2-3 комментария (опционально)

6. АНАЛИТИКА (10 мин)
   ├── Сбор метрик
   ├── Обновление дашборда
   └── Рекомендации

Общее время: ~2-2.5 часа

Правила безопасности

  1. Playwright задачи - ТОЛЬКО последовательно, никогда не параллельно
  2. Human-like задержки: 2-5 сек между действиями, 30-60 сек между платформами
  3. Рандомизация: случайные задержки ±30%
  4. Rate limiting: жёсткие лимиты (см. таблицу), никогда не превышать
  5. Stealth: playwright-extra + stealth plugin
  6. Мониторинг: 3 фейла подряд → СТОП → диагностика
  7. Cooldown: captcha/warning → 24 часа без действий
  8. Ротация: не одинаковые действия в одно время каждый день
  9. Content variety: никогда идентичный текст на 2+ платформах
