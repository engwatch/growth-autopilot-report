Рекомендуемая стратегия подключения

Tier 1: MCP-серверы (моментальная интеграция в Claude Code)

Подключить через claude mcp add, работает из CLI:

  • mcp-twikit (229★) - X без API ключа
  • mcp-linkedin (197★) - LinkedIn фиды + постинг
  • mcp-telegram (239★) - Telegram MTProto
  • jlcases/mcp-reddit (23★) - Reddit с записью!
  • crosspost MCP - 8 платформ одним сервером

Tier 2: Postiz (покрывает 30+ платформ)

Postiz (27 500★) + postiz-agent CLI = один инструмент вместо десяти. Self-hosted Docker.

Tier 3: Прямые API (блог-платформы)

Dev.to REST, Hashnode GraphQL, TabNews REST, Substack python-substack - curl-вызовы из скиллов.

Tier 4: Playwright (нет другого способа)

VC.ru, Хабр, Medium, HN, Threads (комменты) - существующие скиллы Playwright MCP.
