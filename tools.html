Рекомендуемая стратегия

Tier 1: Postiz (покрывает всё)

Postiz (27 611★) + postiz-agent CLI = один инструмент вместо десяти. Self-hosted Docker, 28+ платформ.

Tier 2: MCP-серверы (для отдельных платформ)

mcp-twikit (X), mcp-linkedin, mcp-reddit, mcp-telegram - прямая интеграция в Claude Code.

Tier 3: Прямые API (для блогов)

Dev.to REST, Hashnode GraphQL, Substack python-substack, TabNews REST - curl-вызовы.

Tier 4: Только Playwright

VC.ru, Хабр, Medium (редактирование), HN - нет другого способа.
